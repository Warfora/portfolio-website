### About the website

This is my personal portfolio website, which has professional information about me. The website has the following functionality:
- Home page: displays a video of me paragliding while React's typewriter displays different roles I have
- Projects tab: information, photos, and charts about my personal and school projects. It also displays my GitHub activity using the GitHub REST API
- Contacts tab: contact me by sending an email message, which will be delivered using EmailJS.
- The "Experience" and "About Me" tabs are in progress
- The website is also responsive, so it works well on mobile as well!
